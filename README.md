# Cryptography and Security

This repository contains the completed assignments for the [Cryptography & Security Lab (CSE-4167)](https://drive.google.com/drive/folders/1T8PA_Sw8S_o9CsR8-X5cV4CuYu6bxRYq?usp=drive_link) course at the [University of Dhaka](https://www.du.ac.bd/body/CSE). The assignments cover various cryptographic techniques and security concepts.
